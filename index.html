Project 3
Heart Disease Prediction System

This project entitled “Heart Disease Prediction” intends to use data mining and classification technique for the prediction of likelihood of getting heart disease. It uses Naïve Bayes algorithm that works on the basis of conditional probability. Using medical profiles such as chest pain, exercise induced angina, ST depression, thalium stress test and so on, it can predict the likelihood of patients getting a heart disease. It analyses the performance of the system by calculating the accuracy and other factors using confusion matrix. Similarly, for the user interface, it allows user to input data in the application and provides the corresponding result by calculating the appropriate probability. As well as, the application provides the feature of data visualization through bar graph to the user.

Core technologies used:

  • Naive Bayes Algorithm
  • React Native
  • Javascript
  • Node.js
  • MySql
  • Python
